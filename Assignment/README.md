# FoodHub - Food Delivery Landing Page

A modern, responsive landing page for a fictional food delivery company with all essential sections for a professional service website.

## Features

✅ **Responsive Design** - Works perfectly on all devices (desktop, tablet, mobile)  
✅ **Welcome Banner** - Eye-catching hero section with call-to-action buttons  
✅ **Features Section** - Highlight 6 key features with icons  
✅ **Service Areas** - Display available delivery zones with status  
✅ **Customer Reviews** - Showcase testimonials from satisfied customers  
✅ **Statistics** - Display impressive metrics about the service  
✅ **Contact & Order Form** - Easy way to contact and place orders  
✅ **Newsletter Signup** - Collect customer emails for marketing  
✅ **Professional Footer** - Complete with links and social media  
✅ **Smooth Animations** - Interactive hover effects and scroll animations  
✅ **Modern UI/UX** - Clean design with attractive color scheme

## Customization Tips

### Change Colors
Edit the `:root` variables in `style.css`:
```css
:root {
    --primary-color: #FF6B6B;
    --secondary-color: #FFA502;
    --dark-color: #2C3E50;
    --light-color: #F8F9FA;
}
```

### Update Content
- Replace company name (FoodHub) with your brand
- Update phone number and email in contact section
- Modify service areas and reviews
- Update social media links in footer

### Add Images
Replace SVG placeholders with actual images:
```html
<img src="your-image.jpg" alt="Description">
```
