# Task-7: Modern E-Commerce Store

A responsive e-commerce application built with vanilla HTML, CSS, and JavaScript.

## Features

- **Product Display**: Grid layout showing products with images and prices
- **Shopping Cart**: Dynamic cart system with add/remove functionality
- **Price Calculation**: Real-time total calculation in rupees (₹)
- **Responsive Design**: Works on different screen sizes
- **Modern UI**: Gradient headers, smooth animations, and hover effects

## Structure

- `index.html` - Main HTML file containing structure, styles, and JavaScript

## How to Use

1. Open `index.html` in a web browser
2. Browse products in the grid
3. Click "Add to Cart" to add items
4. View cart on the right sidebar
5. Remove items as needed
6. Click "Checkout" to place order

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Gradients, Animations)
- JavaScript (DOM Manipulation, Event Handling)
- External Images from Unsplash API
